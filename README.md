# VocFilePathReplace

这个工具可以替换voc数据集中xml文件中的图片地址

使用方法：

> main.exe [DataSet Path] [Replace Path]

例子：

> .\main.exe C:\Users\17740\Desktop\DataSet\ D:\xyolo\images\train\

Usage:

> main.exe [DataSet Path] [Replace Path]

For example:

> .\main.exe C:\Users\17740\Desktop\DataSet\ D:\xyolo\images\train\
